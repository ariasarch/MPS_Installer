Getting Started

  • Download: Windows or macOS
  • First run creates a local environment and logs. Keep the Terminal open and watch for progress messages.
  • Project repo: github.com/ariasarch/MPS_1.0.0

Pipeline Overview

  1. Setup & Data Loading (step1_setup.py, step2a_video_loading.py)
  2. Processing & Motion Estimation (step2b_processing.py, step2c_motion_estimation.py, motion_correction.py)
  3. Erroneous Frames & Transform (step2d_erroneous_frames.py, step2e_transformation.py)
  4. Validation & Cropping (step2f_validation.py, step3a_cropping.py)
  5. SVD & Analysis (step3b_svd.py, step3c_svd_analysis.py)
  6. Watershed & Segmentation (step4a_watershed_search.py … step4g_temporal_merging.py)
  7. Noise Est., Validation Setup (step5a_noise_estimation.py, step5b_validation_setup.py)
  8. YRA & Temporal Updates (step6a_yra_computation.py, step6d_update_temporal.py …)
  9. Spatial Update & Merging Validation (step7a_dilation.py … step7f_merging_validation.py)
  10. Final Updates & Save (step8a_final_temporal_update.py … step8c_filter_and_save.py)

Parameters & Files

  • Global parameters: general_processing_parameters.json
  • Parameter storage helpers: parameter_storage.py
  • Utilities & plotting: utilities.py, plotting_functions.py
  • GUI entry: newest GUI_PSS*.py (auto-selected by the launcher)
  • Logs: created under logs/ with timestamps
