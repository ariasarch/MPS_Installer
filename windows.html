Quick Installation Guide

  1. Download the installer above.
  2. Right-click → Run as administrator.
  3. Follow prompts. Git and Conda will be installed if missing.
  4. Package installation may take 15–20 minutes.
  5. Launch MPS via the desktop shortcut.

System Requirements

  • Windows 10/11 (64-bit)
  • 16 GB RAM minimum (32 GB recommended)
  • 10 GB free disk space
  • Stable internet connection
  • Administrator privileges

Troubleshooting (Windows)

  • SmartScreen block: Click More info → Run anyway.
  • Git/Conda errors: Re-run the installer; it will (re)install required components.
  • Slow install: Keep the window open—solves can take time.
  • See Support or open a GitHub issue.
