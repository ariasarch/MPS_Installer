Download for macOS

Signed & notarized launcher app. First run sets up its environment automatically and opens Terminal with live logs.

Get the app

⬇ Download ZIP ⬇ Download DMG

Version 1.0.0 • macOS 12+ recommended • Apple Silicon or Intel

Note: On first run the launcher mirrors its bundled code to ~/Library/Application Support/MPS/, creates a Conda env if missing, and then runs the GUI with full logs.

Quick Start

  1. Download the ZIP (or DMG) above.
  2. ZIP: double-click to unzip → open MPS Launcher.app.
    DMG: open the DMG → drag MPS Launcher.app to Applications (or run in place).
  3. On first launch, if macOS asks “This app was downloaded from the internet,” click Open.
  4. Terminal will appear and show real-time setup/processing logs.

System Requirements

  • macOS 12 (Monterey) or newer
  • 16 GB RAM minimum (32 GB recommended)
  • ~10 GB free disk space for first-run environment install
  • Internet connection for first-run setup

Troubleshooting (macOS)

  • “App can’t be opened” prompt: Right-click the app → Open (one-time). It’s signed & notarized.
  • Conda not found: The launcher will prompt to install Miniforge/Conda; accept and re-run.
  • First run takes a while: Environment solve can take several minutes; watch Terminal for progress.
  • Still stuck? See Support or open a GitHub issue.
