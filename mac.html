Download for macOS

Signed & notarized launcher app. First run sets up the environment automatically and opens Terminal with live logs.

Get the app

⬇ Download ZIP ⬇ Download DMG

Version 1.0.0 • macOS 12+ recommended • Apple Silicon or Intel

Note: On first run the launcher mirrors its bundled code to ~/Library/Application Support/MPS/, creates/repairs a Conda env, and launches the GUI.

Quick Start

  1. Download the ZIP (or DMG).
  2. ZIP: double-click to unzip → open MPS Launcher.app.
    DMG: open → drag MPS Launcher.app to Applications (or run in place).
  3. On first open, if macOS asks “downloaded from the internet,” click Open.
  4. Terminal will appear with setup and processing logs.

System Requirements

  • macOS 12 (Monterey) or newer
  • 16 GB RAM minimum (32 GB recommended)
  • ~10 GB free disk for first-run setup
  • Internet connection for environment install

Troubleshooting (macOS)

  • Can’t open / unidentified developer: Right-click the app → Open once (it’s signed & notarized).
  • Conda missing: The launcher will install Miniforge when needed.
  • Slow first run: Environment solving can take several minutes; watch Terminal for progress.
  • Still stuck? See Support or open a GitHub issue.
