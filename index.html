System Requirements

  • Windows 10 or 11 (64-bit)

  • Stable internet connection for installation

Troubleshooting

If you encounter issues during installation:

  • Git not found: The installer will prompt you to install Git for Windows
  • Conda not found: Choose to install Miniconda automatically when prompted
  • Installation hangs: Package installation takes time - ensure stable internet
  • MPS won't start: Check the terminal output for specific error messages

For detailed troubleshooting, see the installation guide.
